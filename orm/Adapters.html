Adapters #

ORM 内置了对部分数据库的支持. 而通过提供一个适配器, 可以让 ORM 在不同的数据库上运行.

require('@fxjs/orm').addAdapter('customdb', {
  // ...
});

适配器的接口如下:

interface IAdatper {
  /**
   * @description protocol when connecting to database, e.g. 'customdb'
   */
  protocol: string;
}
